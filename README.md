# 🐍 Day 4 – Object-Oriented Programming (OOP) in Python

## 📖 Overview
This assisment demonstrates the core concepts of Object-Oriented Programming (OOP) in Python through practical examples and mini projects. The programs showcase how OOP helps create reusable, scalable, and organized code.

---

## 🎯 Learning Objectives
- Understand Classes and Objects
- Use Constructors (`__init__`)
- Work with Instance and Class Variables
- Implement Encapsulation
- Apply Inheritance
- Demonstrate Polymorphism
- Use Abstraction
- Override Methods
- Understand Access Modifiers

---

## 📚 Topics Covered
- Classes and Objects
- Constructors
- Instance Variables and Methods
- Class Variables
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Method Overriding
- `super()` Function
- Public, Protected, and Private Members

---

## 💻 Programs Included

### 1. Student Management System
- Store student details
- Calculate grades
- Display student information

### 2. Bank Account System (Encapsulation)
- Deposit money
- Withdraw money
- Check account balance

### 3. Employee Management System (Inheritance)
- Parent Class: Employee
- Child Class: Manager

### 4. Vehicle System (Polymorphism)
- Vehicle, Car, and Bike classes
- Demonstrates method overriding

### 5. Shape Area Calculator (Abstraction)
- Abstract Shape class
- Rectangle class implementation

### 6. Library Management System (Mini Project)
- Add books
- Issue books
- Return books
- Display available books

After completing this assignment, you will be able to build modular and reusable applications using Object-Oriented Programming principles in Python.
